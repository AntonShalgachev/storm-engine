Плагин для Maya v4.0, v.4.5

Плагин состоит из 3-х частей.
2 из них Мая-независимые:
export.dll нужно скопировать в папку AW\Maya4.?\bin\
SD2GeoExportMel.mel нужно скопировать в папку AW\Maya4.?\scripts\others\

Эта часть плагина зависит от версии Мая и существует для Маи 4.0 и 4.5
SD2GeoExport.mll нужно скопировать в папку AW\Maya4.?\bin\plugins\

Экспортируемые объекты:
 -полигональный меш.

На объект должен быть назначен материал:
 -Phong
 -Lamber
Incadescene больше нуля для материалов, испускающих световую энергию. Величина - плотность энергии на единицу площади.
cosPow задаёт разброс направлений вылета фотонов относительно нормали вертексов треугольника.

Объект должен содержать мэппинги, к каждому UVSet'у должна быть прилинкована своя текстура.
 -map1 (обязательно) - базовая текстура.
 -bump (опционально) цвет пикселов этой текстуры будет знаково прибавляться к цвету пикселов базовой текстуры.
 -если дополнительному мэппингу не нужен свой UVSet, можно использовать поле Bump Mapping в редакторе материалов Мая.

 Дополнительно, мэппинг учитывает place2dTexture - трансформация текстурных координат:
  Repeat UV - тайлинг текстурных координат,
  Offset - смещение мэппинга,
  Rotate UV - поворот текстурных координат;
все остальные опции в place2dTexture не учитываются.


 Локатор в сцене - указание, что всё, что к нему прикреплено выгрузится в отдельный файл геометрии.
Другие локаторы, к которым прикреплена геометрия, создают отдельный файл - средство образования иерархии.

 Треугольники объекта имеют VertexColor. В Мае изначально vertexColor для вертекса не определён, в этом случае он выгружается со значение ARGB = 255,127,127,127. Цвет вертекса - множитель диффузной компоненты материала. При выгрузке объектов с выкрашенными vertexColor'ами объект нужно триангулировать перед выгрузкой.

 Для выгрузки геометрии необходимо выделить локатор, к которому этот объект привязан. Для выгрузки локаторов без геометрии их нужно также выделить.

 Объект не может быть выгружен, если в материале не назначена текстура.
 Материал с двумя текстурами нельзя класть на объект с одним мэппингом.
 В объекте нельзя сочетать треугольники с одним и с двумя UVSet'ами.

 При выгрузке геометрии параллельно создаются atg-файлы, в которые пишутся найденные ошибки и информация об экспортируемых объектах.
