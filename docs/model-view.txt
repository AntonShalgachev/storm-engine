Утилита для просмотра GM файлов


+ новая фича:
  объект можно импортировать в WaveFront OBJ 




Как пользоватся:
---------------------------------------------


* В окне Select GM file выделите GM файл для просмотра
  (будет выбран оптимальный режим управления камерой и
  файл отобразится в окне ренедра)

 режимов управления камерой два:

   1. Rotate Camera (управление практически аналогично Maya)
       
       Left Mouse   - rotate
       Right Mouse  - scale
       Middle Mouse - Pan
       Mouse Whell  - fast scale


   2. Fly Camera (включается для больших моделей) 

      Переход в режим управления камерой - клик мышью в окне ренедра
      Выход из этого режима ESC или TAB или Middle Mouse

      Управление:
        
       Движение мышью - вращаться
       Left mouse     - вперед
       Right mouse    - назад

      Так же можно использовать клавиши

       Left  = A - стрэйф влево
       Right = W - вперед
       Up    = S - назад
       Down  = В - вправо

      Shift - ускорение
      Ctrl  - ускорение
      
      ускорение с Shift и с Ctrl складываются, т.е. нажав Shift+Ctrl 
      ускорение 2X



* Если в окне Select TX Directory where GM textures placed - выбрана
  директория где лежат текстуры от просматриваемой модели, модель
  будет отображена с текстурами, иначе без

  для отображения модели с текстурами выберите нужную директори и 
  нажминте Reload GM


* В окне Information отображается информация о материалах и текстурах
  используемых в модели

* В окне Weather & Lighting возможно управление освещением и текстурами
  теней для GM файлов которые их имеют

  Для правильной работы отображения текстур теней необходимо правильно 
  указать Resource path (путь где лежат все ресурсы)

  Просматриваемый GM должен находится глубже этого пути

  Двойной клик по погоде (None, Morning, Day, Evening, Storm) перезагрузит
  GM файл используя текстуру с тенями для этой погоды

  Так же можно выбрать файл освещения (COL) для текущей погоды и/или времени
  суток

* В окне Tools находятся различные инструменты для дополнительной работы
  с файлом GM

  Grid On, Grid OFf - соответственно включают и выключают размерную сетку
  (для режима Fly по умолчанию сетка отключена)

  Галочка 3D Grid Mode - расширяет выводимую сетку до 3-х мерной

  Галочка Show Labels - включает отображение названия локаторов

  Галочка Show Locators - включает отображение самих локаторов

* Функция Analyze

   Функция Analyze предусмотрена для автоматического анализа геометрии
   на предмет "висячих" граней у треугольников, т.е. таких граней
   которые не связаны больше не с одним треуголником и образуют "разрыв"
   геометрии

   ПРИМЕЧАНИЕ: Данная функция может выполнятся от 1 до 10 мин. в зависимости
   от конфигурации компьютреа, рекомендуемый объем памяти для данной операции
   256 Мб и выше 
   
   После того, как данная функция отработает результат рассчетов сохраняется
   в кэш файле, который находится в директории с геометрией и имеет аналогичное
   имя с расширением QCH

   Результат работы функции визуально можно посмотреть используя соответствующий
   для этого режим рендера

* Режимы рендера

   В программе предусмотренно несколько режимов рендера:
   информацию о них можно получить по клавише 'H'

   Клавиши для вкл. режимов

    4 - Каркасный режим, показывается wireframe GM файла

    5 - Текстуры + Каркасный режим, поверх обычного рендера накладывается
        wieframe

    6 - Нормальный рендер (геометрия с текстурами)

    7 - Каркасный режим с "висячими" гранями
        аналогично режиму 4, висячие грани выделены красным цветом

    8 - Текстуры + Каркасный режим + "висячие"грани
        аналогично режиму 5, висячие грани выделены красным цветом

    9 - Отключить отображение вертексов

    0 - Включить отображение вертексов


* Дополнительно

   Удерживая клавишу 'G' можно посмтореть информацию о GM файле
    его размер и объем используемой им памяти под текстуры
